How to install the project:

1. Clone the repo:
   ```sh
   git clone https://github.com/Ashley-Nicole18/react-job-portal.git
   ```
2. Install NPM packages:

   ```sh
   cd react-job-portal
   cd backend
   npm install
   cd..
   cd frontend
   npm install
   ```

3. Run the application backend (make sure you are in `/backend` directory) :

   ```sh
   node server.js
   ```

4. Run the application frontend (make sure you are in `/frontend` directory) :
   ```sh
   npm run dev
   ```
5. Open your browser and navigate to `http://localhost:5173` to view the app.


6.  Running Storybook (UI Testing)
   ```sh
   npm run storybook
   ```
